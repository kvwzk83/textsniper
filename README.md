## 用起来是什么感觉

装好之后，菜单栏会多一个小图标，平时完全无感知。需要抓文字的时候，按 `Shift-Command-2`（比系统截图快捷键少一个数字，挺好记的），像截图一样框一下，松手瞬间文字就已经在剪贴板里了。整个流程行云流水，用过一次就懂为什么大家都说"用了就离不开"。

**关键一点：完全离线运行。** 不联网，不上传截图，所有识别都在本地完成。对于处理合同、报告、私密文档的人来说，这个特性尤其重要——你的内容不会经过任何第三方服务器。

## 功能不止识别文字

TextSniper 能做的事，比"图片转文字"要多一些：

**屏幕 OCR 识别**：图片、视频截帧、PDF、网页、应用界面……只要屏幕上有文字，就能框出来。视频里的字幕、演示文稿里的内容、甚至游戏界面里的文本，全都能抓。

**多语言支持**：除了简繁中文和英文，还覆盖德语、法语、日语、韩语、西班牙语等多种语言。macOS 14 Sonoma 及以上版本还额外支持竖排文字识别，做日语、古汉语相关工作的用户直接受益。

**追加剪贴板**：做读书笔记时，可以连续框选多段文字，TextSniper 会自动把几次识别结果拼在一起，一次性粘贴，不用来回切换。

**文本转语音**：识别完的文字可以直接朗读出来。学外语时遇到不认识的词，框一下就能听到发音，比切到翻译软件快多了。

**二维码 / 条形码识别**：屏幕上出现二维码，不用掏手机扫，直接框选就能提取里面的信息或链接。

**iPhone / iPad 联动**：可以调用手机摄像头扫描纸质文件，联动识别，不用专门找扫描仪。

## 跟系统自带的 Live Text 比呢

macOS Monterey 以后，苹果自带了 Live Text，也能识别图片里的文字，免费。那 TextSniper 有必要买吗？

差别主要在几点：Live Text 只能识别照片和 Safari 里的图片，TextSniper 能识别屏幕上任何东西，包括视频、应用界面、锁定 PDF；TextSniper 的识别速度更快，快捷键更灵活；追加剪贴板、文本转语音、二维码识别这些功能，系统自带的都没有。

如果只是偶尔需要抓一两段文字，Live Text 够用。但如果经常要处理 PDF、视频内容、图片文档，TextSniper 的效率提升相当明显。多位媒体评测都提到，它的识别准确率高于 Adobe Acrobat 和 Google Docs 的内置 OCR。
